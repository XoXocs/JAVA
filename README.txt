Java Programs Collection
Author: Anirudh

This folder contains multiple Java programs covering basic to intermediate concepts.

List of Programs:
1. FibonacciSeries.java - Prints Fibonacci numbers up to N terms.
2. Factorial.java - Calculates factorial of a number.
3. PrimeCheck.java - Checks if a number is prime.
4. PalindromeCheck.java - Checks if a string is a palindrome.
5. TrafficSignal.java & Main.java - Simulates a round-robin traffic light system.

How to Run:
- Compile: javac FileName.java
- Run: java FileName
